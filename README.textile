h2. Habrahabr — Inversion compact skin

Стиль делает Хабр более уютным и удобным:
- компактная шапка
- меню пользователя на правой стороне
- уменьшены размеры показателей кармы/рейтинга/индекса
- заменил появляющиеся точки около комментариев на отступ размером в аватарку + тоненькую линию для определения уровня вложенности
- заменил множественные "ответить" на скромный линк, который появляется при mouse hover
- убрал округленные уголки с аватарок
- если срок голосования за комментарий истек, то показывается только рейтинг, без серых кнопок
- убавил ширину правой колонки

"Топик обсуждение стиля":http://habrahabr.ru/post/131992/
